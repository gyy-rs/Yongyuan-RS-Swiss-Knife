# TROPOMI SIF Spatial Downscaling (ESA–MOST Dragon Programme)

## 1. Scientific Abstract

This module implements a proof-of-concept framework for spatially downscaling coarse-resolution TROPOMI solar-induced fluorescence (SIF) to finer spatial resolutions using high-resolution optical indices. Developed as part of the **ESA–MOST Dragon Programme (2025)**, the workflow explores how machine learning can bridge the scale gap between TROPOMI footprints (kilometre scale) and Landsat/Sentinel-2 pixels (tens-of-metres scale). The primary goal is to resolve sub-pixel variations in vegetation functioning and moisture stress, while acknowledging that the project focus subsequently shifted towards a dedicated Xinjiang experimental site.

**Status**: Archived prototype (research sandbox).

---

## 2. Technical Implementation

- **Random Forest downscaling model**
  - Uses a Random Forest regressor to learn a mapping from high-resolution spectral/vegetation indices to TROPOMI-like SIF.
  - Trained on spatially homogeneous “pure pixels” where aggregation from fine to coarse scale is well-defined.
  - Employs a log(1 + y) transform of target SIF for stabilising variance and improving regression behaviour.

- **Feature engineering and data cleaning**
  - Ingests point- or pixel-level samples containing SIF and reflectance-based predictors.
  - Cleans NaN and infinite values from both features and targets.
  - Removes negative target values to avoid issues with logarithmic transforms.

- **Model evaluation and visual diagnostics**
  - Splits data into training and test subsets.
  - Evaluates performance using RMSE and R² on both transformed and original scales.
  - Produces publication-quality plots of observed vs. predicted SIF using Seaborn and Matplotlib.

- **Raster-based prediction workflow**
  - Applies the trained Random Forest model to multi-band GeoTIFF stacks (e.g. Landsat/Sentinel-2 derived features).
  - Handles NoData masks and preserves georeferencing and metadata.
  - Writes high-resolution predicted SIF rasters for downstream mapping and analysis.

---

## 3. Key Files Description

- `rf_downscaling_modeling.py`  
  Model development and evaluation pipeline:
  - Loads tabular training data with SIF and predictor features.
  - Configures plotting style and colour palettes for scientific visualisation.
  - Cleans the dataset, applies log(1 + y) to the target, and fits a tuned Random Forest in log space.
  - Evaluates the model on a held-out test set and plots observed vs. predicted SIF.
  - Outputs performance metrics and figures summarising model skill.

- `rf_downscaling_prediction.py`  
  Inference pipeline for generating high-resolution SIF maps:
  - Loads a trained scikit-learn model from disk (`.joblib`).
  - Iterates over all GeoTIFF files in a user-specified directory.
  - Validates the number of raster bands against the expected feature list (`FEATURE_COLUMNS`).
  - Applies a NoData mask, reshapes valid pixels to (n_samples, n_features), and runs batch predictions.
  - Reconstructs full-size output rasters, fills non-valid pixels with NoData, and saves predicted SIF maps as GeoTIFFs.

## 4. Usage / Example

### 4.1. Model training and evaluation

```bash
cd 03_tropomi_sif_downscaling
python rf_downscaling_modeling.py
```

This script:
- Loads the prepared training dataset (see paths inside the script).
- Trains the Random Forest downscaling model in log space.
- Evaluates performance on a test split and exports a scatter plot of observed vs. predicted SIF.

### 4.2. Apply the trained model to GeoTIFF feature stacks

```bash
cd 03_tropomi_sif_downscaling
python rf_downscaling_prediction.py
```

Before running:
- Set `MODEL_PATH` to the `.joblib` model generated by the training script.
- Configure `INPUT_DIRECTORY` to point to multi-band GeoTIFF feature stacks.
- Configure `OUTPUT_DIRECTORY` to a directory where predicted SIF maps will be written.
- Ensure that `FEATURE_COLUMNS` exactly matches the order and semantics used during model training.

The script will generate new GeoTIFF files with suffix `_predicted_SIF.tif`, representing spatially downscaled SIF estimates suitable for exploratory mapping and hypothesis testing.
